## ① テキストの"受け取り方"の心理学（なぜ実際よりキツく届くのか）

```
あなたは対人コミュニケーションおよびCMC(computer-mediated communication)を専門とする社会心理学の研究者です。「テキスト（チャット/メール/コメント）が、送り手の意図より"キツく・ネガティブに"受け取られるのはなぜか、そして受け手側の解釈は何に左右されるか」を、査読付き研究を優先して調査してください。

明らかにしたい問い（各々エビデンスの強さを評価）:
1. 非言語・パラ言語手がかりの欠落が解釈に与える影響。テキストでトーン/皮肉/善意が失われるメカニズム
2. ネガティビティ・バイアス、および「曖昧なメッセージは受け手の気分が低い時ほどネガティブに解釈される」等の気分一致処理の証拠
3. 送り手が「自分のトーンは伝わる」と過信する現象（例: Krugerらのemail egocentrism/過信研究）と、受け手との認識ギャップ
4. 受け手の状態（ストレス・疲労・心のキャパ・自己肯定感）が解釈の辛辣さをどう変えるか
5. 送り手の地位・序列・関係の親密度/履歴が「刺さり方」に与える影響（power/familiarity効果）。非匿名の職場文脈での増幅
6. 高文脈/低文脈など文化差、特に日本語のコミュニケーション規範における誤読の起き方
7. 誤読を減らすために「不足している文脈を補う」ことの有効性の根拠

出力: メカニズム一覧（エビデンス強度つき）→ 受け手側の状態要因 → 送り手/序列の効果 → 文化・日本語特有の論点 → 反証/異論 → 「キツく感じたテキストに対し、落ちている文脈を補い善意に再解釈する支援ツール」への示唆。出典URL付き、英語・日本語ソース併用、査読研究を優先。
```

## ② 感情への"対処"：エビデンス／臨床ベース（医学的に何が効くか）

```
あなたは臨床心理学・感情制御(emotion regulation)を専門とする研究者です。「対人的な一言に傷ついた/カッとなった瞬間、後悔する反応を避け、心理的ダメージを減らすには、医学的・臨床的に何が有効か」を、査読付き・臨床エビデンスを優先して調査してください。

明らかにしたい問い（各々エビデンスの強さを評価）:
1. Grossの感情制御プロセスモデル: 認知的再評価(reappraisal)・抑制(suppression)・注意そらし・受容 の効果と副作用の比較（reappraisalが有効/suppressionが高コスト、の根拠）
2. CBTの認知の歪み（読心・破局化・過度の一般化）を「メッセージ解釈」に当てはめた再構成(reframing)の有効性
3. 怒り/衝動のマネジメント: 生理（扁桃体ハイジャック、感情の波は数十秒〜90秒等の主張の妥当性）、間を置く/タイムアウト、呼吸法、urge surfing(DBT)の証拠
4. 苦痛耐性と感情の妥当化(validation, DBT): なぜ「まず気持ちを正当と認める(validate)」方が、いきなり再解釈するより効果的か。無効化(invalidation)の害の証拠
5. 対人的な軽視/失礼への反応低減に、reappraisal が機序として効くという直接的エビデンス
6. 個人差・逆効果になる条件（認知負荷が高い/キャパが低い時に技法が破綻するか）

出力: エグゼクティブサマリ → エビデンスに基づく対処技法のランキング（強度つき）→ 「validate → reappraise」の順序原則 → 逆効果の条件 → 「傷ついた瞬間に押すと、気持ちを認めた上で再解釈と冷静な返信を渡すボタン」が"何を・どの順で・どんな言葉で"返すべきかの示唆。出典URL付き、英語・日本語ソース併用、査読/臨床研究を優先。
```

## ③ 介入の設計とリスク：リフレーミング系デジタル介入は"効く・害さない"か

```
あなたはデジタルメンタルヘルスおよびHCI・行動科学を専門とする研究者です。「その場でユーザーの感情/解釈を再構成する短時間のデジタル介入は、実際に有効か、そしてどんな条件で害になるか」を、査読付きエビデンスを優先して調査してください。対象は2015年〜2026年。

明らかにしたい問い（各々エビデンスの強さを評価）:
1. 短時間のreappraisal/reframing介入の有効性（"別の読み方をしてみて"型の介入は本当に苦痛を減らし、より良い対人反応につながるか）
2. Just-in-time adaptive intervention(JITAI): 「まさにその瞬間」に出す介入のタイミング設計と効果条件
3. AI/チャットボットによる感情サポート・再解釈の効果と**リスク**: 過度の依存、感情回避/対処スキルの衰え(deskilling)、toxic positivity、無効化(invalidation)、そして「本物の人間接触をAIが間に挟んで遠ざける」害
4. 職場文脈: 心理的安全性・コミュニケーション風土。ツールが機能不全を"助ける"のか"覆い隠す"のか
5. 倫理・設計ガードレール: 同意、ガスライティングを避ける、抑制/回避を助長しない、ユーザー状態への適応
6. 効果測定: "気分が良くなった"だけでなく、実際に助けになっているかをどう測るか

出力: エグゼクティブサマリ → 短時間再構成介入の有効性エビデンス → 効く条件(JITAI等) → **明確な失敗モード（依存・回避・無効化・過剰仲介）と設計での回避策** → 職場・倫理の論点 → 「受け手の防衛ボタン」を"効果的かつ安全"にするための出力/UX/ガードレール設計への具体的示唆。出典URL付き、英語・日本語ソース併用、査読研究を優先。
```
